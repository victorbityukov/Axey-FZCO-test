## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
$ yarn install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# watch mode with .env.dev
$ npm run start:dev:env

# production mode
$ npm run start:prod
```
## API

```
http://localhost:3000
```

## Swagger

```
http://localhost:3000/docs
```
